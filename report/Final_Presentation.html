Introduction & Problem Statement

The Problem

  • Small and medium businesses in India struggle with manual bookkeeping
  • Spreadsheet-based tracking is error-prone and hard to scale
  • Existing solutions (Tally, QuickBooks) are expensive or too complex
  • No unified platform for transactions, invoicing, clients, and reporting
  • Lack of GST-compliant record keeping for Indian businesses

Our Solution: COUNTORI

  • A modern, web-based bookkeeping application
  • Simple, clean interface designed for non-accountants
  • Tracks income, expenses, invoices, clients, and vendors
  • Real-time financial dashboard with visual charts
  • GST-ready with GSTIN and PAN support for all entities
  • Accessible from any device with a browser

Project Objectives

Transaction Management

Record and categorize all income and expenses with full CRUD

Invoice Generation

Create, track, and manage professional invoices with status workflow

Client & Vendor Records

Maintain a database of business contacts with outstanding balances

Financial Reporting

Visual dashboard with Profit & Loss, Balance Sheet, and Cash Flow

Implementation: Middleware & Authentication

Custom Middleware (src/middleware.ts)

Runs before every request:

  • Logs request method, path, URL, timestamp
  • Calculates response duration
  • Injects custom headers (X-Countori-Middleware)
export function middleware(request: NextRequest) {
  // [BEFORE] Log incoming request
  console.log(`[BEFORE] ${method} ${path}`);

  const response = NextResponse.next();

  // [AFTER] Log response
  console.log(`[AFTER] ${path} ${duration}ms`);

  response.headers.set(
    'X-Countori-Middleware', 'active'
  );
  return response;
}

JWT Authentication (src/lib/auth.ts)

Secure token-based login flow:

  • Passwords hashed with bcryptjs (12 rounds)
  • JWT tokens signed with secret key (7-day expiry)
  • Stored in HTTP-only cookies (XSS-safe)
  • Protected routes check auth on every load
// Password hashing
bcrypt.hash(password, 12);

// JWT token generation
jwt.sign(payload, SECRET, {
  expiresIn: '7d'
});

// Secure cookie storage
cookieStore.set('countori_token', token, {
  httpOnly: true,
  secure: true,
  sameSite: 'lax',
});

export async function PUT(request, { params }) {
  const { id } = await params;
  const body = await request.json();
  const transaction =
    store.updateTransaction(id, body);
  if (!transaction)
    return NextResponse.json(
      { error: 'Not found' }, { status: 404 }
    );
  return NextResponse.json({
    success: true, data: transaction
  });
}
